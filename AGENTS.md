# Agent Instructions

This repository's development guidelines for AI coding agents live in
**[`CLAUDE.md`](CLAUDE.md)**. The conventions there apply to all agents
(Claude, Codex, Cursor, Copilot, Gemini, etc.), not just Claude — read it
first.

Subsystem-specific instructions, when present, live in
`internal/<subsystem>/CLAUDE.md` (for example,
[`internal/v3net/CLAUDE.md`](internal/v3net/CLAUDE.md)). Read the relevant
subsystem file before working in that area; it takes precedence over the
general guidelines for its scope.
